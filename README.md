# Custom Browser Main Page - Web (Next.js Version)

Esta es la nueva versión de la landing page para el proyecto [Custom-Browser-Main-Page](https://github.com/AgustinBeniteez/Custom-Browser-Main-Page).

Construida con **Next.js 14**, **Tailwind CSS**, **Framer Motion** y **Lucide React**. Optimizada para ser desplegada en **Vercel**.

## 🚀 Características

- 🎨 **Diseño Moderno**: Interfaz limpia con gradientes y animaciones suaves.
- 📱 **Responsive**: Totalmente adaptada para dispositivos móviles y escritorio.
- 📜 **Notas de Parche Dinámicas**: Cargadas directamente desde un archivo JSON.
- 📖 **Documentación**: Sección dedicada para guiar a los usuarios y desarrolladores.
- ⚡ **Performance**: Optimización de imágenes y carga estática con Next.js.

## 🛠️ Tecnologías

- [Next.js](https://nextjs.org/) (App Router)
- [Tailwind CSS](https://tailwindcss.com/)
- [Framer Motion](https://www.framer.com/motion/)
- [Lucide React](https://lucide.dev/)

## 💻 Desarrollo Local

1. Instalar dependencias:
   ```bash
   npm install
   ```

2. Ejecutar servidor de desarrollo:
   ```bash
   npm run dev
   ```

3. Construir para producción:
   ```bash
   npm run build
   ```

## 📂 Estructura del Proyecto

- `/src/app`: Páginas y layouts de la aplicación.
- `/public`: Activos estáticos (imágenes, iconos).
- `/legacy`: Versión anterior de la web (HTML/CSS estático).
